###web前端-如何学习web前端~

###web前端学习时间安排

可以照着妙味课堂的视频里面的代码敲，一步一步来，练熟练度，找感觉
http://2017.miaov.com/study

###初级阶段-16天

1.熟悉常用术语—2天
web前端
h5移动端
html
css
javascript-js
jquery
等等，在网上多搜一些，理解这些简称的含义

相关资料查看，好好看一下大牛的总结
https://github.com/qiu-deqing/FE-learning

2.html/css/javascript基础知识-5天

可照着写代码，把文件另存为.html结尾的文件，可以直接用浏览器打开，初步感受一下写代码的乐趣

  • html相关-1天
    • http://www.runoob.com/html/html-tutorial.html
  • css相关-1天
    • http://www.runoob.com/css/css-tutorial.html
  • js相关-3天
    • http://www.cnblogs.com/jikey/p/3613082.html

html和css的代码照着敲就好了，写多了就理解了，
javascript是非常重要的，重点学习

3.深入理解html/css/javascript-2天

  • html5
    • http://www.runoob.com/html/html5-intro.html
  • css3
    • http://www.runoob.com/css3/css3-tutorial.html
  • js
    • http://www.liaoxuefeng.com/wiki/001434446689867b27157e896e74d51a89c25cc8b43bdb3000

简单来说，html5和css3是最新的标准

4.理解和掌握jquery-2天

http://www.runoob.com/jquery/jquery-tutorial.html

5.理解Ajax-1天

看看就好了，后面再掌握

http://www.cnblogs.com/dolphinX/p/3240927.html

6.掌握js和html、css在浏览器下的调试方法-2天

7.理解json数据格式，接口交互方式-2天

###中级阶段

这一个阶段主要是学习常用框架和插件的使用
学会自己在网上找代码

  • 框架:http://www.bootcss.com/ http://jqweui.com/

  • 插件:http://datatables.club/ https://validator.niceue.com/examples/

    • http://www.oschina.net/news/56950/jquery-bootstrap-plugins-for-your-next-projects (bootstrap 插件)
