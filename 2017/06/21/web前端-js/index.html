###web前端-js学习相关-重要点

全面的学习可参考下面这个网站
http://www.w3school.com.cn/h.asp

js虽然叫做浏览器脚本语言，也包含了代码语言的设计思想

javascript的学习要点如下：
1.js实现

2.js语句

3.js注释

4.js变量

5.js数据类型

6.js对象

7.js函数

8.js运算符，比较，if…else for while等

DOM-文档对象模型(其实就是html里面的各个标签，js可以获取到这些标签，改变这些标签的属性)

掌握了js的基本使用之后，需要什么函数查文档就好了

实际在使用的过程汇总，肯定要用javascript框架库

重点学习JQuery
jquery本质还是JavaScript，只不过他封装了很多底层的实现，极大的简化了js的开发，学习不难，就是要多练习
1.jquery语法
2.基本的jquery的效果

AJAX也是学习的重点，是与服务器交换数据的艺术，它在不重载全部页面的情况下，实现了对部分网页的更新。
在不重载整个网页的情况下，AJAX 通过后台加载数据，并在网页上进行显示。
AJAX = 异步 JavaScript 和 XML（Asynchronous JavaScript and XML）。
