3.背景 background
1）.背景颜色：background-color:gray;
2）.背景图片：background-image：url(bg.jpg/gif);
3）.背景重复：background-repeat：no-repeat/repeat-x/repea-y/repeat；
4）.背景位置：background-position：X轴px Y轴px （X轴位置：left center right Y轴位置：top center bottom）

4.文字的设置
1）.文字粗细：font-weight:bold(加粗)；normal（正常）;
2）.文字倾斜：font-style：italic(斜体) normal（正常）
3）.文字大小：font-size（一般均为偶数）
4）.行 高：line-height（文字在一行里所占用的位置 当行高的值一致时文字会垂直居中显示）
5）.多行文字的测量方法：
A．确认文字位置
B.确认两行文字之间的空隙大小
C.空隙大小除以2.得出来的值就是每行文字的上下空隙大小。当行 为奇数时，文字的上方会比下方少一个像素，为偶数时一致
D.通过辅助项测量多行文字的行高
6）.字 体：font-family（中文默认宋体）
符合样式font需要注意书写的顺序：font-style font-weight font-size
3.文本设置
1）.文字颜色：color
2）.文本对齐方式：text-align
3）.首行缩进（em缩进字符）：text-indent
4）.文本修饰：text-decoration
5）.字母间距：letter-spacing
6）.单词间距（以空格为解析单位）：word-spacing
7）.强制不还换：white-space
Nowrap(不换行) normal（正常/换行）
8）.一个空格有多大？
字体格式为宋体时，空格大小为当前文字大小的一半
字体格式不一样时，空格大小也不一样

2017.6.23 学习小结
1.padding
Padding是内填充 设置padding后会撑大原本容器的大小
pading的四个方向：top right bottom left
单独设置一个“padding”属性值时，四个方向都是同一个属性值
设置两个属性值时，第一个属性值设置的是上下两个方向，第二个属性值设置的是左右两个方向
设置三个属性值时，第一个属性值设置的是上方向，第二个是左右两个方向，第三个则是下方向的属性值
设置四个属性值时，第一个属性值设置的是上方向，第二个是右方向，第三个则是下方向的属性值，第四个值是左方向（顺时针）。

2.margin 外边距
1）.Margin是外边距。指的是标签与标签之间的间隔（距离）。
Margin的四个方向：top right bottom left
单独设置一个“margin”属性值时，四个方向都是同一个属性值
设置两个属性值时，第一个属性值设置的是上下两个方向，第二个属性值设置的是左右两个方向
设置三个属性值时，第一个属性值设置的是上方向，第二个是左右两个方向，第三个则是下方向的属性值
设置四个属性值时，第一个属性值设置的是上方向，第二个是右方向，第三个则是下方向的属性值，第四个值是左方向（顺时针）。
2）.Margin存在的问题：
A．marigin-top的传递
解决方法：目前已学知识点Border可以解决传递问题
B．margin上下叠压
解决技巧：a.使用margin，可以将某个元素方向设置为预想的值
Marigin叠压则会取最大值
b.如果元素没有特殊特征。是可以用padding代替的

3.盒模型
1）.盒模型大小：border+padding+width/height
2）.盒模型宽度：border-left-width+padding-left+padding-right+border-right-width+width
3）.盒模型高度：border-top-width+padding-top+padding-bottom+border-bottom-width+height

2017.6.24 学习小结
1.伪类：给文字添加特殊的效果

  • hover 鼠标移入（悬停）
  • link 未访问过的链接的初始颜色
  • active 鼠标按下时链接的颜色
  • visited 访问过后链接的颜色
    伪类顺序： L V H A

2.a标签（超链接）
1）.a标签：超链接
2）.span标签：区分样式
3）.header标签：页眉。用于页面的头部的信息介绍，也可以用于板块头部
4）.nav标签：导航（包含链接的一个列表）
5）.footer：页脚。用于页面的底部或者板块的底部
6）.section页面上的板块 用于花粉页面上不同区域或者划分文章里不
同的节
7）.article标签：用来在页面中表示一套结构完整且独立的内容部分
8）.aside标签：元素标签可以包含与当前页面或者主要内容相关的引
用、侧边栏、广告、nav元素组，以及其他类似的有别于主要内容 的部分
9）.time：用来表现时间或者日期
3.常用标签
1）.h1-h6 标题
2）.P 段落
3）.strong 强调作用（粗体）
4）.em 强调作用（斜体）
5）.span 区分样式
6）.ul 无序列表
7）.ol 有序列表
8）.li 列表项
9）.dl 定义列表
