###美利金融研发组织架构,产品线梳理

美利金融在北京，上海，深圳均有研发团队

1.深圳研发

  • 1.前端

    • web前端(h5移动端) : 负责有用分期web端，h5端的前端(包括To c 面向用户的产品，内部系统等)
    • app(android,ios) : 负责有用分期app的开发

  • 2.后台(全部是java)

    • 信贷业务组

      • 3c线下贷研发 : 手机，电脑，摩托车，驾校分期
      • 线下现金贷
      • 销售产品
      • 商户产品

    • 风控组 : 征信平台，风险控制团队，对接聚信立，同盾，国政通，百融，爰金，等数据公司

    • 催收，反欺诈组 : FI管理，催收系统

    • 电销，客服组 : 电话营销，客服

    • 业务效能组

    • 大数据组 : 数据报表，数据分析，数据建模，知识图谱，数据查询平台

    • 信审系统 : 进件-审核-生成合同-导标-还款计划

    • 基础服务平台 : 通知平台，影像平台，权限系统

  • 3.测试团队

  • 4.产品团队

2.北京研发

  • 支付系统

  • 结算系统

  • 融资-资产系统

  • 负债系统

  • 财务系统

  • 新信审系统

  • 有用商城

  • 有用现金贷PDL(pay day loan)

3.上海研发

  • 车联盟

  • 车商城

  • 车直销
