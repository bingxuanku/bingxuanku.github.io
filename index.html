mac搭建个人博客-hexo+github

重点：所有的hexo命令都需要加sudo

1.安装node.js,git
2.安装hexo
sudo npm install -g hexo
3.进入自己创建的一个文件夹(比如myblog)，初始化hexo
sudo hexo init
4.进入myblog这个文件夹
生成静态页面 sudo hexo generate
本地启动 sudo hexo server
5.新建文章
sudo hexo new “文章名”
会在source/_posts/下新建一个 .md的文件，打开这个文件，编写文章即可
6.推送到github上
先绑定githug账号
打开_config.yml文件
在最下面添加
deploy:
type: git
repository: https://github.com/bingxuanku/bingxuanku.github.io.git
branch: master

在打开http://bingxuanku.github.io/ 查看效果
7.替换皮肤
下载：sudo git clone https://github.com/A-limon/pacman.git themes/pacman
在_config.yml指定皮肤
编辑文件_config.yml，找到theme一行，改成 theme: pacman
然后静态化处理，再发布到github
